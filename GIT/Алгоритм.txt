Как достать файлы
git clone https://github.com/Wandish/My_Telebot.git
Посмотреть всех веток в т.ч. удаленных
git branch -a
Переключение на ветку
git checkout

Как отправить файлы
git add * или .
git commit -m "name"

Отправить на удаленный  репозиторий
git push origin main 
Получить с удаленного репозитория
git pull origin main

Чтобы вернуться назад, нужно перейти на коммит, после чего сделать перемещение (выйти с коммита)и там закомититься.
